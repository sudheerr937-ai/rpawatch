# Contributing to RPAWatch

Thank you for your interest in contributing!

## How to contribute

1. Fork the repository
2. Create a feature branch: `git checkout -b feature/your-feature`
3. Make your changes
4. Test locally
5. Submit a pull request

## Areas where contributions are welcome

- Additional Orchestrator API metrics
- FISMA/FedRAMP compliance logging enhancements
- Support for older Orchestrator API versions
- Additional remediation workflow templates
- Documentation improvements

## Reporting issues

Open a GitHub issue with:
- Your Orchestrator version (Cloud/On-Prem)
- Your UiPath plan type
- The error message and logs
